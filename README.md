# 📄 Resume Screening System

AI-powered Resume Screening System built using Streamlit and NLP techniques.

## 🚀 Overview

This application analyzes resumes against a job description using a Hybrid Scoring approach combining:

- Semantic Similarity (50%)
- Skill Category Match (30%)
- Mandatory Skill Coverage (20%)

It provides ranking, decision support, and skill gap analysis.

---

## ✨ Features

- 📊 Hybrid Scoring Algorithm
- 🔍 Mandatory Skill Extraction
- 📈 Skill Category Breakdown
- ⚠️ Keyword Gap Analysis
- 🏆 Resume Ranking System
- 💡 Explainability Panel

---
